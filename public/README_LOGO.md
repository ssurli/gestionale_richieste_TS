# Logo USL Toscana Nord Ovest

Per visualizzare correttamente il logo nell'applicazione:

1. Salva il logo dell'USL Toscana Nord Ovest come: `logo-usl-tno.png`
2. Posiziona il file in questa directory (`public/`)
3. Il logo verrà automaticamente visualizzato nell'header

## Specifiche Logo

- **Nome file**: `logo-usl-tno.png`
- **Formato consigliato**: PNG con sfondo trasparente
- **Dimensioni consigliate**: Larghezza minima 200px, altezza 80-100px
- **Posizione**: `/public/logo-usl-tno.png`

## Fallback

Se il logo non viene trovato, il sistema utilizzerà automaticamente un'icona di fallback (FileText).
